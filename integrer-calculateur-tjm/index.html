Code d’intégration du widget

Copiez ce bloc dans le code HTML de la page où vous souhaitez afficher le simulateur. Le cadre occupe la largeur disponible, jusqu’à 640 pixels.

<style>.tjmcalc-embed{width:100%;max-width:640px;border:0;border-radius:12px}.tjmcalc-credit{margin:.4rem 0;font:12px/1.4 sans-serif}.tjmcalc-credit a{color:#047857}</style>
<iframe class="tjmcalc-embed" src="https://tjmcalc.com/?embed=1" width="640" height="900" loading="lazy" title="Simulateur TJM vers salaire net freelance"></iframe>
<p class="tjmcalc-credit"><a href="https://tjmcalc.com/">Calculateur TJM gratuit par TJMCalc</a></p>
Astuce : le bouton « Copier le code d’intégration » présent sous le calculateur principal fournit exactement le même bloc.

Ce que le widget permet de calculer

Le lecteur peut partir de son TJM ou d’un revenu net cible, puis modifier son activité, son statut, ses jours facturés, ses frais et ses principales hypothèses fiscales.

Chargement adapté aux pages de contenu

L’attribut loading="lazy" diffère le chargement tant que le widget est loin de la zone visible. Les calculs sont exécutés dans le navigateur ; les valeurs saisies ne sont pas envoyées à un formulaire TJMCalc.

Mises à jour centralisées

Les améliorations du calculateur sont disponibles automatiquement dans le cadre intégré. La méthodologie et les sources officielles permettent d’expliquer ses hypothèses.

Conserver le lien d’attribution

L’intégration est proposée gratuitement avec la mention visible « Calculateur TJM gratuit par TJMCalc ». Conservez ce lien afin que les lecteurs puissent vérifier la méthode, les sources et la version complète de l’outil.

N’utilisez pas le widget pour présenter une estimation comme un conseil comptable, fiscal ou juridique personnalisé.
