Signaler une correction

Pour qu’une correction soit vérifiable, indiquez l’URL concernée, la valeur contestée, la date d’observation et, si possible, un lien vers une source officielle.

Les propositions de nouvelles fourchettes métiers doivent préciser s’il s’agit d’une expérience individuelle, d’un jeu de données ou d’une étude documentée.
